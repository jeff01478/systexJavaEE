# 樂透號碼產生器(lottery) & 猜數字遊戲(guessGame)

## 功能
* lottery：隨機生成排除指定號碼後的6個1~49之間的樂透數字。

* guessGame：在限定的次數內猜1~10之間的數字

## 使用說明
### 1. lottery
#### 輸入要的組數以及想排除的數字後點選送號
> 兩者皆不可空白且需是大於0的數字
